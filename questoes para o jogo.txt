a - ULA (unidade lógica e aritmética)

c - CPU (central de processamento) - resposta certa
d - PC (contador de programa)

a - ULA (unidade lógica e aritmética) - resposta certa
b - UC  (unidade de controle)
c - CPU (central de processamento)

a - ROM
b - RAM
c - Barramento de dados
d - Registradores - resposta certa


4 - São tipos de memórias os seguintes itens:

a - RAM, ROM, Eprom, Flash, memória em massa. - resposta certa

c - RAM, ROM, UC, PC, memória de massa.
d - Eprom, Flash, memória de massa.

c - PC
d - Barramento de endereço

a - CS (chip select) - resposta certa
b - UC (unidade de controle)
c - PC (contador de programa)
d - Barramento de componentes

7 - Os barramentos transportam informações entre os vários componentes internos do processador, como cache L1, registros. Seguindo essas definições, para barramento de endereço e barramento de dados podemos chamá-los também:
